## 1. Introduction

This document defines common terms and abbreviations used across the FS-AI ADS-DV documentation set.

It is intended to be used alongside the current User Manual, Operational Safety, Transportation Procedures, Inspection Schedule, RAMS, and other related ADS-DV documents.


## 2. Scope and Use

This glossary provides the preferred definitions and terminology for shared operational, safety, transport, and project terms.

Where a document contains interface-specific, competition-specific, or software-specific terminology, that document may still define additional local terms. In particular, the [FS-AI ADS-DV Software Interface Specification](./FS-AI%20ADS-DV%20Software%20Interface%20Specification.md) remains the authoritative source for fixed CAN identifiers, message names, and interface-specific abbreviations.


## 3. Glossary

| Term                              | Definition / Use in Current Docs                                                                                        |
| --------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| ADS-DV                            | Autonomous Driving System - Dedicated Vehicle.                                                                          |
| ADS-DV Operator                   | The responsible person or organisation using the ADS-DV and accountable for planning, documentation, preparation, and overall control. |
| AI Power Switch                   | The switch used to supply power to the Compute Platform. |
| ASR                               | Autonomous Systems Responsible person. The responsible operator who supervises the ADS-DV and operates the RES.         |
| Autonomous System Master Switch (ASMS) | The switch controlling the Autonomous System.                                                                      |
| CAN                               | Controller Area Network, the interface type used for the ADS-DV communications.                                         |
| Compute Platform                  | Generic term for any hardware used to send CAN signals to the ADS-DV.                                                   |
| DCDC Converter                    | Direct Current / Direct Current converter used to recharge the LV Battery from the Traction Battery.                    |
| Dynamic Operating Area            | The designated area, pre-approved by the IMechE, within which Dynamic Operation is permitted.                           |
| Dynamic Operation                 | Use of the ADS-DV within an approved Dynamic Operating Area under the defined ASR and area-control rules.               |
| EBS                               | Emergency Braking System.                                                                                               |
| E-Stop Loop                       | Electrical circuit completed through the correct state of the RES, Side Emergency Stop Switches, MPP Dongle, and Inertia Switch. |
| GEMS DA3 Datalogger               | The GEMS DA3 data-logging unit used to record ADS-DV run data.                                                          |
| GEMS DA3 Datalogger Memory Card   | The removable memory card used by the GEMS DA3 Datalogger.                                                              |
| IMechE                            | Institution of Mechanical Engineers.                                                                                    |
| InCarPC                           | A specific type of Compute Platform optionally used by some ADS-DV Operators.                                           |
| LV Battery                        | Low Voltage Battery.                                                                                                    |
| Mission                           | The selected autonomous run or task chosen through the Touch Screen interface before activating the RES "Go" signal.    |
| MPP                               | Multi-Purpose Port.                                                                                                     |
| MPP Dongle                        | The dongle fitted to the MPP required for operation and Brake Discharge.                                                |
| Quick Lift Jack                   | The approved jack used to lift the ADS-DV on the underside of the chassis.                                              |
| RAMS                              | Risk Assessment & Method Statement.                                                                                     |
| RES                               | Remote Emergency Stop. The emergency stop device operated by the ASR.                                                   |
| Run Off Zone                      | The defined 10m boundary around the internal operating area of the Dynamic Operating Area.                              |
| Safe Area                         | The area for general personnel during Dynamic Operation, separated from the Dynamic Operating Area by a structural barrier. |
| Skateboard Trolleys               | The supplied wheeled supports used to carry or support the ADS-DV.                                                      |
| Static Operation                  | Use of the ADS-DV in a workshop, enclosed space, or in the Safe Area adjacent to an approved Dynamic Operating Area.    |
| Tractive System Master Switch (TSMS) | The switch controlling the tractive system.                                                                          |
| TSAL                              | Tractive System Active Light. The indicator LEDs used to show tractive-system and autonomous-state conditions.          |
| VCU                               | Vehicle Control Unit.                                                                                                   |
