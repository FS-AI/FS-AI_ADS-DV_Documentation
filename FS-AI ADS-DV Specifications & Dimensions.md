## 3. Drive System

The drive-system specification is as follows:

- Drive Motor: `Saietta 119R-68`
- Peak Motor Torque: `55.7Nm`
- Maximum Motor Speed: `4,000rpm`
- Belt Drive Ratio: `3.5:1`
- Motor Controllers: `Sevcon Gen4 DC`
- Maximum Electrical Power: `5kW`


## 4. Steering System

The steering-system specification is as follows:

- Steering Motor: `DC Electronics Microsteer`
- Steering Controller: `Shiftec ZE102-1`


## 5. Traction Battery

The traction-battery specification is as follows:

- Battery Chemistry: `Lead Acid`
- Cell Capacity Rating: `100Ah`
- Pack Configuration: `12V x4`
- Nominal Pack Energy Capacity: `5.10kWh`
- Nominal Pack Voltage: `50.0V`
- Operating Voltage Range: `48.0-52.0V`
- Peak Discharge Current: `200A`
- Continuous Discharge Current: `100A`


## 6. Dimensions & Configuration

The key dimensions and configuration values are as follows:

- Wheelbase: `1530mm`
- Front Track: `1200mm`
- Rear Track: `1200mm`
- Overall Width: `1430mm`
- Overall Length: `2815mm`
- Static Caster: `6`
- Static Camber: `2`
- Driven Wheels: `RWD`
- Differentials: `Open Diff`
- Tyres: `Toyo Proxes R888R 225/45 R13 84V`
